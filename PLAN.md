# 🧸 토이팡팡 (ToyPangPang) - 최종 기획 및 테스트 명세

## 1. 프로젝트 개요
- **서비스명**: 토이팡팡 (ToyPangPang)
- **목표**: 실시간 홈쇼핑 감성을 담은 어린이 장난감 전문 쇼핑몰.

## 2. 핵심 사용자 시나리오 (User Journey)
1. **탐색**: 메인 페이지에서 실시간 타이머와 '매진임박' 상품을 확인한다.
2. **상세 확인**: 관심 있는 상품을 클릭하여 상세 정보를 읽는다.
3. **담기**: 상품을 장바구니에 추가하고 헤더의 카운트가 올라가는 것을 확인한다.
4. **주문**: 장바구니 페이지에서 수량과 총액을 확인하고 '주문하기'를 누른다.
5. **완료**: 결제 완료 페이지를 확인하고 홈으로 돌아온다.

## 3. 테스트 전략 (Testing Strategy)
- **빌드 테스트**: `npm run build`를 통해 모든 페이지(정적/동적)의 무결성 검증.
- **로직 테스트**: `CartContext`의 추가/삭제 기능 및 로컬 스토리지 유지 기능 확인.
- **반응형 테스트**: 모바일(360px) 및 데스크탑(1200px) 레이아웃 대응 여부 확인.

## 4. 최종 배포 사양
- **Framework**: Next.js 16 (App Router)
- **Style**: CSS Modules, Global Variable Root
- **State**: React Context API
